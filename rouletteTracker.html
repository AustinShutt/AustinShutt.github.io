.NET MAUI

Roulette Tableside Tracker Application


This application is being built in C# using Microsoft's MAUI framework to develop a roulette application tracker for Android, Iphone, and possibly Windows.

The code for this project can be found on this github page.

Design Requirements

Purpose:
Create a mobile application that allows a user to track roulette spin progress and be able to view the statistics of the betting session in a clear and easy to understand way.

Requirements:

  1. User able to select the last winning number from the roulette table.
  2. User able to delete the record beginning with most recent entry.
  3. User able to clear the session data.
  4. Display history data in running list breaking down between red/black even/odd low/high.
  5. Display history data in graphs between competing bets.

Moon Shot Ideas:

  • Track wins and losses during a single betting session.
  • Recommend current bet amount according to win/loss history and progression betting system.
  • (Really out there and maybe legal risk) Recommend next bet according to win/loss history and table history.

Expected Challenges:

  • How to support both an American Wheel and European Wheel versions of roulette.
  • Most efficient way to draw desired graphs to the screen.

Implementation

This project will be a collaborative effort working with a programming partner heavily utilizing the tools of GitHub, Miro, and Discord in order to communicate our goals and progress throughout the project which will ultimately culminate in posting to the Android, IOS, and Windows App stores.

Technical Mockup

This mockup was built using Xamarin forms utilizing the mvvm design pattern with data binding.
